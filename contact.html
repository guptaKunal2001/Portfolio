Research Paper: A Novel FBit_Sort Algorithm for digital encoding process

This research paper has proposed Fbit_sort algorithm that enables the user to sort the data available in binary floating form.
The proposed algorithm for Binary data can be used in digital encoding process on which computer language and programming is based on.
In addition to this, the sorting algorithm can avail the user of the application of binary numbers in Boolean Algebra, a branch of mathematics.

Methodology

  • The proposed Fbit_sort algorithm is a novel one, that can be used to efficiently manage and sort the binary floating data entered by the user.
