Quiz and it's Admin App

Description:

    Front-end:

    • Home screen: Display a list of categories available to the user.
    • Category selection screen: Display the sets available for the selected category.
    • Set selection screen: Display the questions available in the selected set.
    • Question screen: Display the question with four options and a timer.
    • Result screen: Display the final score.

    Back-end:

    • Implement a database to store categories, sets, and questions.
    • Create API endpoints to fetch category lists, sets, and questions based on user selection.
    • Implement a scoring mechanism to calculate the user's score.
    • Handle user responses and validate correct answers.

Technologies Used:

  • XML
  • Java
  • Android Studio
  • Firebase
